READ ME

(1)External Libraries Used
Besides the dns libraries (except for the dns.resolver), python sys and time libraries	are used.

(2) mydig.py is for resolving type "A", "NS" and "MX" domains. Two arguments are needed, website' domain name and query type.

(3) mydig_dnssec is for both IP resolving and validating. Two arguments are needed, website' domain name and query type.
